It's time to start talking about color.
How could you do the first five lessons without?

    color( r, g, b);

There are several colorsystem available in P5. In the RGB-systems, the different colors are created by mixing light. When you mix red light (r) with green light (g) and blue light (b) you will get white light. The r,g and b can have a value between 0 and 255 and again color(0,0,0) is black and color(255,255,255) is white.

Several functions you already know, can use the same params as the color function:

  • background( 100, 40, 35)
  • fill( 220, 0, 255)
  • stroke( 60, 0, 0)

In the previous lesson you learned about variables. You can use variables to store your color values. 

        var color1 = color( 255, 0, 0);

Now red is stored in the variable color1 and can be used else where. The color values are often set in the setup function and used in the draw function, when you change these values in the setup, all the colors will changes in the drawing. 

  var color1, color2, color3;         //declaration

  function setup(){
    createCanvas(500,500);
    background(100);
    color1 = color(0);                //they need a value,
    color2 = color(180);              //before you can use them.
    color3 = color(100,0,0);

    frameRate(5);
  }

  function draw(){
    stroke(color3);
    strokeWeight(3);
    fill(color2)
    rect(random(width),random(height),
         random(50,100),random(10,50));  //you can write
                                         //a function over two lines.
    //javascript will ignore the white space.

    stroke(color3);
    strokeWeight(5);
    fill(color1)
    rect(random(width),random(height),
         random(10,50),random(50,100));

  }
