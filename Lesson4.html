Lines are wonderful. You can do so much with lines.

  • Forms, Shapes, Figures
  • Patterns
  • Shading
  • Perspective and space
  • Optical illusions

In this lesson we will look at the straight lines only, but these straight lines are graphically interesting, especially when you start repeating these lines. We can write every line down in the drawing function, but it is much smarter to use a variable.

Variables

A variable is like a 'box' where you can store a value.
If you create a variable in a function, the variable is only known in this specific function.
Often we need to give a variable a value before we start a function.
In this case we declare the variable outside the function, before the setup function. 

    var x;          //declaration of the variable x;
    function setup(){
      createCanvas(400,400);
      x = 0;        //x needs a value before you can call it in the draw function.
    }
    function draw(){
      line(x, 200, 400, 200);
      x += 2;       //add two more in your 'box'
    }

You can create a straight with the line function.
You always need two points to create a line.

      line(x, y, x1, y1);

The line function has four params:

  • the first two params are the begin-position (x,y) of the line
  • the third and the fourth params are the end-position(x1,y1) position of the line.

the params (x,y) mark the center of the point. The line need the same drawing tools as the point. You need a stroke function, so your drawingtool knows which color to use. You need a strokeWeight function, to tell your drawingtool how big your dot should be.

The beginning and the end of the line can be set outside the canvas. 

  function setup(){
    createCanvas(300,300);
    background(100);
  }

  function draw(){
    stroke(240);
    strokeWeight(30);
    line(85, 130, 200,155);

    stroke(160);
    strokeWeight(20);
    line(50, 230, 200,-10);

    stroke(80);
    strokeWeight(10);
    line(40, 250, 240,160);

    stroke(0);
    strokeWeight(1);
    line(5, 290, 180,200);
    line(180,200, 100,20);
    line(100,20,5,290);
  }

1. P5 Reference

  • Go to the reference op P5 at p5js.org/reference
  • Search for the line method
  • You can find every function or method back in the Reference.
  • In openprocessing go to the menu on the right side of your screen. Next to the tab Setting you find two more tabs Files and Reference. Click on the Reference tab and click on a functionname in the list. You will find the P5 reference page about this function.

2. One_Line_Drawing

  • create a canvas as wide as your window and 400 pixels height.
  • First make a sketch on graph paper. It will save you a lot of time.
  • create a sketch in the same size, where 50 pixels is 1 cm.
  • Use a variable x for the begin points and end points of you lines.

    • line(x + 0, 300, x + 50, 300);
    • line(x + 50, 300, x + 150, 100);
    • etc.
  • After you have declared your x variable outside all the functions, you can start with x = -width in the setup function;
  • Now the line can move to the right, if we add a speed value to the x.
