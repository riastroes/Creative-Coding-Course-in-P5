Lines are wonderful. You can do so much with lines. You can create

  • Forms, Shapes, Figures
  • Contours
  • Patterns
  • Shading
  • Perspective and space
  • Optical illusions

In this lesson we will look at the straight lines only, but these straight lines are graphically interesting, especially when you start repeating these lines. We can write every line down in the drawing function, but it is much smarter to use a variable.

  • line
  • p5 Reference

You can create a straight with the line function.
You always need two points to create a line.

      line(x, y, x1, y1);

The line function has four params:

  • the first two params are the begin-position (x,y) of the line
  • the third and the fourth params are the end-position(x1,y1) position of the line.

Lines needs the same drawing tools as the point. You need a stroke function, so your drawingtool knows which color to use. You need a strokeWeight function, to tell your drawingtool how width your line should be.

The beginning and the end of the line can be set inside or outside the canvas.

In the next lesson as we learn more about variables you can start enjoying repeating lines on a smart way. Now you have to draw indidual lines, so make them special. 

  function setup(){
    createCanvas(300,300);
    background(100);
  }

  function draw(){
    stroke(240);
    strokeWeight(30);
    line(85, 130, 200,155);

    stroke(160);
    strokeWeight(20);
    line(50, 230, 200,-10);

    stroke(80);
    strokeWeight(10);
    line(40, 250, 240,160);

    stroke(0);
    strokeWeight(1);
    line(5, 290, 180,200);
    line(180,200, 100,20);
    line(100,20,5,290);
  }
